1. Information We Collect

This system is designed for personal use and does not collect personal information from external users. The system processes:

  • Media files (images and videos) uploaded for posting
  • Generated captions and hashtags
  • Posting schedules and queue information

2. How We Use Information

Information processed by this system is used solely for:

  • Automated posting to connected social media platforms
  • Generating AI-powered captions and hashtags
  • Managing posting schedules and content queues
  • Storing media files temporarily for processing

3. Data Storage

Media files are stored temporarily in AWS S3 for processing and posting. Files are automatically cleaned up after processing. No personal data is permanently stored.

4. Third-Party Services

This system integrates with the following third-party services:

  • Social Media Platforms: Instagram, TikTok, Tumblr, Bluesky
  • Cloud Services: AWS S3 (storage), AWS Bedrock (AI captions)
  • Development Platform: GitHub (code hosting and automation)

Each service has its own privacy policy that governs how they handle data.

5. Data Security

We implement appropriate security measures to protect data during processing:

  • API credentials are stored securely using environment variables
  • All communications with third-party APIs use secure HTTPS connections
  • Temporary files are automatically deleted after processing

6. Data Retention

Media files are retained only as long as necessary for processing and posting, typically less than 24 hours. Queue and scheduling data is retained for operational purposes but contains no personal information.

7. Personal Use Only

This system is designed for personal use by the developer. No data is shared with third parties beyond what is necessary for posting to the connected social media platforms.

8. Your Rights

Since this is a personal automation system, traditional data subject rights may not apply. However, if you have concerns about data processing, please contact us via the GitHub repository.

9. Changes to This Policy

This privacy policy may be updated from time to time. Any changes will be reflected in the "Last Updated" date below.

10. Contact Information

For questions about this privacy policy or data processing practices, please contact via the GitHub repository linked above.
